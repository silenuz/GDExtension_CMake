Introduction
------------

The `godot-cpp-template <https://github.com/godotengine/godot-cpp-template>`_ is a quickstart template for GDExtension
development with `Godot <https://godotengine.org/>`_ .

This introductory module consists of a walkthrough covering the creation process of a GDExtension using CMake, from
acquiring the template to extending the extension with custom functionality.

The first steps section details the steps necessary to acquire the template, as well as cmake configuration, an initial build,
and test.

The extension configuration section details the necessary steps to rename the library, and the edits necessary to
accommodate the new name.
